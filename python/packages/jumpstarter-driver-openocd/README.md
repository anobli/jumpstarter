# OpenOCD Driver

`jumpstarter-driver-openocd` provides functionality for interacting with openocd devices.

## Installation

```shell
pip3 install --extra-index-url https://pkg.jumpstarter.dev/simple/ jumpstarter-driver-openocd
```

## Configuration

Example configuration:

```yaml
export:
  openocd:
    type: jumpstarter_driver_openocd.driver.OpenOCD
    config:
      # Add required config parameters here
```
